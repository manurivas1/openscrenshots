# Designer Skill - Examples & Demos 🏷️

This document catalogs high-impact designs that can be generated using the **Designer Skill**.

## 🚀 Hero Showcase
A primary screen designed for the App Store.

- **Theme**: `Indigo Dark`
- **Layout**: `generateBottomStack`
- **Focus**: High contrast, premium tilted device.

## 📱 Features List
A list-based design for explaining app functionality.

- **Theme**: `Minimal Mint`
- **Layout**: 3 Centered Devices side-by-side.
- **Focus**: Cleanliness and readability.

## 🌅 Sunset Lifestyle
Media-heavy app showcase.

- **Theme**: `Sunset Warm`
- **Layout**: Overlapping devices with soft shadows.
- **Focus**: Vibrant colors and engagement.
